# 🗄️ Data Digger - E-Commerce SQL Database Project

## 📌 Description
Data Digger is a SQL-based database project that simulates an e-commerce system. It manages customers, products, orders, and order details while performing various operations like data insertion, updates, deletions, and analytical queries.

## 🎯 Objective
The objective of this project is to design a relational database and perform real-world SQL operations such as CRUD operations, joins, aggregations, and data analysis.

## 🧱 Database Structure
The database consists of the following tables:
- Customers
- Products
- Orders
- OrderDetails

Each table is connected using foreign key relationships to maintain data integrity.

## 🚀 Features
- Create and manage database schema
- Insert sample data
- Perform CRUD operations (Create, Read, Update, Delete)
- Use constraints (PRIMARY KEY, FOREIGN KEY, CHECK)
- Execute analytical queries
- Calculate total order amounts dynamically
- Perform joins between multiple tables

## 🛠️ Technologies Used
- SQL (MySQL)
- Relational Database Design
